== Description ==

Stop the automatic formatting with articles and widgets so that <br /> tags are automatically inserted in line breaks.
記事やウィジェットで自動整形を停止し、改行に自動で<br />タグが挿入されるようにします。

== Installation ==

1. Upload `tijaji-formatting` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set it from customizer.For details, check "tijaji-formatting" in the setting menu.
